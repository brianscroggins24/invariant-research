# invariant-research

`invariant-research` is the research wiki for the `invariant-engine` trading project.

It stores research, synthesis, decisions, experiments, and agent context that are useful for understanding the project and for producing reviewed context packs for `invariant-engine`.

It does not store live prices, orders, positions, balances, or production risk state. Those belong in `invariant-engine` and its runtime systems.

The repository is intentionally structured to evolve from ingested sources while keeping governance boundaries stable.
